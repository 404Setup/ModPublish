# Overview

Welcome to the ModPublish documentation. ModPublish is an open-source suite of tools designed to streamline the process of publishing Minecraft mods to multiple distribution platforms.

## Core Purpose

Publishing mods can be repetitive and time-consuming. You often have to export JAR files, log into multiple sites, copy/paste changelogs, select compatible loaders, and check MC versions individually. ModPublish automates this in a GUI-driven manner inside your IDE, instantly pushing files to Modrinth, CurseForge, GitHub, and GitLab.
