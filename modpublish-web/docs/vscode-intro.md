# VS Code Edition

A lightweight, highly responsive Visual Studio Code port of the ModPublish tool suite.

## Features

* **Explorer Context Menu Integration:** Right-click on any compiled `.jar` or `.litemod` file in your workspace, and select "Publish Mod..." to instantly open the publishing UI.
* **Vanilla JS Webview:** Built completely with native JS, HTML5, and CSS variables. It inherits your current editor theme colors (Light, Dark, High Contrast) seamlessly.
* **Instant Metadata Reading:** Scans the jar archives locally to parse dependency info, loaders, and versions within milliseconds.
* **Git Integration:** Dynamically pre-fills branch and repo details.

## Installation

Search for **"Minecraft Mod Publish Utils for vsc"** in the VS Code extensions market, or download from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=404Setup.modpublish-vsc).
