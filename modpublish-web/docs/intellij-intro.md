# IntelliJ IDEA Edition

Quickly publish your Minecraft mods to multiple mod hosting sites directly from IntelliJ IDEA.

## Key Features

* **Fully GUI-based operations:** Fill and check details, then publish in one-click.
* **Automatic Metadata Extraction:** Auto-reads mod information from fabric.mod.json, mods.toml, neoforge.mods.toml, etc., saving manual inputs.
* **History Retention:** Remembers your last-configured dependencies and changelogs.
* **Simultaneous Uploads:** Publishes mods to Modrinth, CurseForge, GitHub, and GitLab simultaneously.
* **Built-in Description Syncing:** Syncs or updates mod descriptions and info pages directly from local Markdown files.

## Installation

Install "Minecraft Mod Publish Utils" from the JetBrains Marketplace:

1. Open IntelliJ IDEA.
2. Go to **Settings** (or Preferences) > **Plugins**.
3. Search for `Minecraft Mod Publish Utils` in the Marketplace tab.
4. Click **Install** and restart the IDE.
5. Alternatively, download directly from the [JetBrains Marketplace Page](https://plugins.jetbrains.com/plugin/28320-minecraft-mod-publish-utils).
